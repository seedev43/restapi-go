## API Reference

#### Ephoto360

```http
  GET /ephoto/?text1=hello&text2=world&url=https://en.ephoto360.com/create-an-online-team-logo-in-black-and-white-style-689.html
```

| Parameter | Type     | Description  |
| :-------- | :------- | :----------- |
| `text1`   | `string` | **Required** |
| `text2`   | `string` | Optional     |
| `url`     | `string` | **Required** |
